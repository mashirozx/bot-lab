About

Mastodon 是一个基于 ActivityPub 和 OStatus 协议的类似推特、微博的联邦式社交网络。它提供的开放式的 API 允许使用者接入到 Mastodon 的几乎每一个工作流程中，因此，我们可以通过 API，创造一些外部脚本/工具以实现特定功能。在 Mastodon 世界中，有一类账号，他们由外部程序控制，我们称之为 BOT（机器人）。
目前小森林上就运行着多个机器人，你可以在后面看到它们的列表，它们的源代码大部分已经开源在 GitHub 上，你可以自由地使用到任何一个 Mastodon 实例上。

Open Source

bili2toot

自动将 Blibili 动态搬运到 Mastodon 的程序。主要功能：正文格式化、媒体文件（图片、动图）搬运。
演示：@arknights

tweet2toot

自动将 Twitter 搬运到 Mastodon 的程序。主要功能：正文格式化、媒体文件（图片、视频、动图）搬运。
演示：@ArknightsStaff
